# Página Principal Back

Backend desarrollado con NestJS, Prisma, MySQL y JWT para la administración de la página principal.

## Tecnologías

- NestJS
- TypeScript
- Prisma
- MySQL
- Docker
- JWT
- Bcrypt

## Requisitos

- Node.js
- Docker Desktop
- npm
